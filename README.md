# Rolligan

A push-your-luck dice party game for groups of 2–10 players. Roll the dice, build the pot, bank your points before someone rolls a 7.

Built with Vite + React. Deployed on Vercel.

© 2026 Clarendon Labs LLC

## Local dev

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run preview
```
